# Machine Learning Models using Decision Tree and Linear Regression

2. Weather Prediction using Linear Regression (R² Score)

## Datasets
- Loan Approval Dataset (US & Canada)
  https://www.kaggle.com/datasets/parthpatel2130/realistic-loan-approval-dataset-us-and-canada

- Szeged Weather Dataset
  https://www.kaggle.com/datasets/budincsevity/szeged-weather

## Models Used
- Decision Tree Classifier (Gini Index)
- Linear Regression (Least Squares Method)

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- Classification Report
- Gini Index
- R² Score
